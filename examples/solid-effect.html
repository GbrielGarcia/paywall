Solid Effect

This page demonstrates the solid effect. The overlay creates a solid color that darkens gradually over time.

Solid Red Overlay
<script src="node_modules/paywall-js/dist/paywall.min.js"></script> <script> new Paywall({ dueDate: '2025-07-15', daysDeadline: 10, effect: 'solid', color: '#ff0000' }); </script>

Configuration

  • effect: 'solid'
  • color: '#ff0000' (solid color)
  • dueDate: '2025-07-15'
  • daysDeadline: 10

How it works

  • Creates a solid color overlay using the specified color
  • Overlay darkens as payment becomes more overdue
  • After 10 days, overlay reaches maximum opacity
  • Includes subtle "Pago Pendiente" message
  • Simple and effective for urgent payment reminders

Color Examples

  • Red: '#ff0000' - Urgent payment
  • Orange: '#ff6600' - Warning
  • Blue: '#0066cc' - Professional
  • Purple: '#6600cc' - Premium
