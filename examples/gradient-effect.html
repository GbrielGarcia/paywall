Gradient Effect

This page demonstrates the gradient effect. The overlay creates a beautiful gradient that intensifies over time.

Red to Black Gradient Overlay
<script src="node_modules/paywall/dist/paywall.min.js"></script> <script> new Paywall({ dueDate: '2025-07-15', daysDeadline: 10, effect: 'gradient', gradientFrom: '#ff0000', gradientTo: '#000000' }); </script>

Configuration

  • effect: 'gradient'
  • gradientFrom: '#ff0000' (starting color)
  • gradientTo: '#000000' (ending color)
  • dueDate: '2025-07-15'
  • daysDeadline: 10

How it works

  • Creates a gradient overlay from gradientFrom to gradientTo
  • Gradient intensifies as payment becomes more overdue
  • After 10 days, gradient reaches maximum opacity
  • Includes subtle "Pago Pendiente" message
  • Uses mix-blend-mode: multiply for better visual effect
