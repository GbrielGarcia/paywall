Simple Usage

This page shows the default gradient effect in action.

Gradient Effect Active
<script src="node_modules/payment-protection/dist/payment-protection.min.js"></script> <script> new PaymentProtection({ dueDate: '2025-07-15', daysDeadline: 10 }); </script>

Configuration

  • dueDate: Payment due date (YYYY-MM-DD)
  • daysDeadline: Days after which effect is maximum
  • effect: 'gradient' (default), 'solid', 'fade'

How it works

  • If payment is overdue, effect appears
  • Effect intensifies as days pass
  • After deadline, effect is maximum
