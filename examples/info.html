What is Paywall?

Paywall is a simple JavaScript library that helps developers ensure they get paid by gradually making websites unusable when payments are overdue.

// Basic usage new Paywall({ dueDate: '2025-07-15', daysDeadline: 10 });

Configuration Options

Option Type Default Description
dueDate String Required Payment due date (YYYY-MM-DD)
daysDeadline Number 60 Days after which effect is maximum
effect String 'gradient' Effect type: 'gradient', 'solid', 'fade'
color String '#ff0000' Color for solid effect
gradientFrom String '#ff0000' Starting color for gradient
gradientTo String '#000000' Ending color for gradient

How it works

  1. Set Configuration: Define your payment due date and deadline
  2. Automatic Check: Library checks if payment is overdue
  3. Visual Effect: Applies chosen effect based on days overdue
  4. Gradual Intensification: Effect becomes stronger over time
  5. Maximum Impact: At deadline, effect reaches full intensity
